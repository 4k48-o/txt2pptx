# 概览

> 在 Manus 中直接访问内置的第三方数据

## 如何使用数据集成（Data Integration）

你只需要在 prompt 中描述你需要的数据，Manus 会自动选择并使用合适的数据源。

## 说明

<Info>
  所有数据源都包含在你的 Manus 订阅中，无需额外的 API key 或配置。
</Info>

<Note>
  数据可用性与速率限制会因提供方不同而异，也可能受额外限制影响。Manus 会自动处理 API 调用与缓存，以获得更佳的性能表现。
</Note>

<Warning>
  第三方数据受其服务条款与数据新鲜度策略约束。对于关键结论，请务必从官方来源交叉验证。
</Warning>
